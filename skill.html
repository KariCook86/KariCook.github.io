Creating HTML files

The contents of a website are what is being displayed on an html file. This is ultimately the page(s) that contain images, multimedia, text, and other content. The html file is loaded into the browser and shows the content of each element but without a stylesheet the content would be plain, text only and non-responsive.

CSS Stylesheet files

The CSS stylesheet(s) contains the majority of styling for the content displayed in the html files. Styling is where design and creativity is at the hands, or fingers, of the creator. Each element of a html page can be made unique, responsive, interactive and ultimately creating a professional website.

Responsive and Professional

Not all devices have the same screen width so there is a need to create responsive pages. The user will be who judges and interacts with the page and everyone has their preferred device. Professional looking websites will reflect the professionalism of a business, entity or person(s).
